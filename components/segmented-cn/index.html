何时使用

  • 用于展示多个选项并允许用户选择其中单个选项；
  • 当切换选中选项时，关联区域的内容会发生变化。

配合 name 使用

可以为 Segmented 配置 name 参数，为组合内的 input 元素赋予相同的 name 属性，使浏览器把 Segmented 下的 input 真正看作是一组（例如可以通过方向键始终在同一组内更改选项）。
